## 🛠️ How to Use

### Prerequisites
- Java JDK 17 or higher
- GCC/G++ compiler for C++ files
- Python 3.x

### Running Java Solutions
```bash
# Compile any Java file
javac SolutionName.java

# Run the compiled class
java SolutionName
```

### Running C++ Solutions
```bash
# Compile any C++ file
g++ SolutionName.cpp -o solution

# Run the executable
./solution
```

### Running Python Solutions
```bash
# Run any Python file directly
python3 SolutionName.py
```
